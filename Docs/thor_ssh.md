# Thor SSH

### Instructions for connecting to the Nvidia Jetson Thor development board using ssh from a linux computer

1. Be on same network 
   (Eudoram will likley prohibit ssh-tunnel to be established, use LAN or mobile hotspot)

2. ip-adress to thor (private at home network, not applicable at school): 192.168.0.183

3. Connect via ssh using the following command:

    ```bash
    ssh thor@192.168.0.183
    ```
